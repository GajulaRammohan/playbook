## Stylus

We have not yet chosen a style guide for Stylus, other than to decide that:

* We *don't* use `;`.
* We *do* use `{` and `}`.
* We *do* indent with 2 spaces.
* We *do* format class names in `dash-case`
* We *do* format file names with `dash-case`.
